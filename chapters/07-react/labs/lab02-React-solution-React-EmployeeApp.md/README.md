# React Employee Dashboard Solution

This folder contains a complete frontend solution for the Employee Management Application lab.

## Structure

- `package.json`
- `public/index.html`
- `src/index.js`
- `src/App.jsx`
- `src/services/api.js`
- `src/pages/HomePage.jsx`
- `src/pages/AddEmployeePage.jsx`
- `src/pages/EditEmployeePage.jsx`
- `src/components/EmployeeList.jsx`
- `src/components/EmployeeCard.jsx`
- `src/components/EmployeeForm.jsx`

## Instructions

1. Open this folder in a terminal.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the app.
4. The app will allow you to list, add, edit, and delete employee records.
