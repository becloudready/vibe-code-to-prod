# React Product Catalog Solution

This folder contains a complete frontend solution for the React Product Catalog lab.

## Structure

- `package.json`
- `public/index.html`
- `src/index.js`
- `src/App.jsx`
- `src/ProductCatalog.jsx`
- `src/ProductFilters.jsx`
- `src/ProductList.jsx`
- `src/ProductCard.jsx`
- `src/index.css`

## Instructions

1. Open this folder in a terminal.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the app.
4. Use the search input, category filter, and stock toggle to filter products.
