# React Todo Dashboard Solution

This folder contains a complete frontend solution for the React Todo Dashboard lab.

## Structure

- `package.json`
- `public/index.html`
- `src/index.js`
- `src/App.jsx`
- `src/TodoApp.jsx`
- `src/TodoForm.jsx`
- `src/TodoList.jsx`
- `src/TodoItem.jsx`
- `src/index.css`

## Instructions

1. Open this folder in a terminal.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the app.
4. The app allows adding, toggling, deleting, and filtering todos.
